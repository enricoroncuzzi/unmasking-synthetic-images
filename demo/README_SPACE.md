# Unmasking Synthetic Images — Forensic MoE Demo

Detect whether an image was generated by an AI model and identify which
diffusion model produced it (Stable Diffusion 1.5 / 2.1, SDXL Base, SD 3.5, FLUX.1).

**Architecture:** five ResNet50 experts (each fine-tuned on a specific SD variant)
feed into a Mixture-of-Experts gating network. The gating weights (α) reveal
which expert "recognises" the image, attributing it to a specific generator.

| Metric | MoE-Logit | MoE-Embedding |
|---|---|---|
| Balanced Accuracy | 94.1% | 95.2% |
| AUC-ROC | 0.988 | 0.990 |

**Attribution:** the α bar chart shows the expert routing weights — a high weight
on the FLUX expert indicates FLUX-generated content, and so on.

**Grad-CAM:** visualises where the attributed expert's attention concentrates
to make its decision.
